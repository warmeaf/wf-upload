# 介绍

## wf-upload 是什么？

wf-upload 是一个基于文件分片的大文件上传解决方案。主要包含以下 3 个部分：

1. 前端实现
2. 通讯协议
3. 基于 nest.js 的后端实现

## wf-upload 解决了什么问题？

1. 网络中断、程序异常退出等问题导致文件上传失败，从而不得不全部重新上传
2. 同一文件被不同用户反复上传，白白占用网络和服务器存储资源

## wf-upload 和其它大文件上传方案的区别

1. 全栈实现，不仅仅是前端，还包括通讯协议，以及基于 nest.js 的后端实现
2. 支持文件流式下载
3. 文件分片贯穿上传到存储的整个过程，不存在任何数据冗余
